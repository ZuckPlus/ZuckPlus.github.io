Week 3: Exploring Fastify

Fastify

Fastify is a web framework for Node.js that prioritizes speed and low overhead. It is known for its efficiency, scalability, and ability to handle requests.

Why did I choose Fastify?

Compared to Express.js, Fastify provides improved performance due to its schema-based validation and optimized handling of requests. It is also highly extensible with its plugin architecture. I personally compared the speed between Fastify and Express.js with making a python script that locally enumlated users. I sent multiple requests from rotating proxies and there is a clear difference between Fastify and Express.

How to start.

To set up Fastify, install it using npm:

npm install fastify

Then, create a simple server:


const fastify = require('fastify')();
fastify.get('/', async (request, reply) => {
    return { message: 'Hello, Fastify!' };
});
fastify.listen({ port: 3000 }, err => {
    if (err) throw err;
    console.log('Server running on http://localhost:3000');
});
