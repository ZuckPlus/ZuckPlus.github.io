Week 7: Learning PostgreSQL

PostgreSQL

This week, I started learning PostgreSQL as a step towards mastering more advanced database management systems like Firebase and Supabase. PostgreSQL is an open-source, and extensible SQL database that is widely used in production environments.

It provides strong ACID compliance, advanced indexing techniques, and support for complex queries. Many companies use firebase or supabase as their database or a form of their database.

Basic Queries & Table Setup

Creating a table in PostgreSQL is similar to other SQL databases but with additional features.

PostgreSQL supports JSON fields, full-text search, and advanced indexing, which makes it a strong candidate for future scalability.
